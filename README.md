# About

<p align="left">
  <img src="assets/logo-flat.png" width="250" align="right"/>
	The <b>Calisthenics Alliance (CAL)</b> is an endurance-focused calisthenics league featuring head-to-head battles between athletes. Organized by the <i>Associação B.W.C.R. — Clube de Calistenia das Caldas da Rainha</i> in collaboration with other clubs and athletes from across the national calisthenics community, CAL was created to establish a structured, continuous, and fully regulated competitive system for endurance calisthenics. The league consists of multiple events held throughout the season, where athletes face off in one-on-one battles, culminating in a grand final that determines the season’s champion.
</p>

# Installation

This project is a simple static website (HTML, CSS, JavaScript), built with NextJS.

- Make sure you have Node.js (which includes npm and npx) installed.

```sh
npm i           # Install the npm packages locally
npm run start   # Start the local server
```

- Once the server is running, open: [http://localhost:3000](http://localhost:3000)
